Latest version from GitHub

The GitHub repository is located at https://github.com/InverseLight/ValoMC/ .

Typing

git clone https://github.com/InverseLight/ValoMC/

at command prompt will obtain the latest version directly from the GitHub repository. It will create a folder named 'ValoMC' in the directory the command was invoked at. Note that you can also use the GitHub repository to report an issue, create pull requests etc.
